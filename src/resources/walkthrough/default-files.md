## Open Default Files

Some folders come with **default files** — the files you'll use most often.

To open them quickly:

1. **Right-click** a folder in the Explorer
2. Select **Memoria: Open default file(s)**
3. Your default files open side by side

For example, right-clicking `00-ToDo/` opens `Main.todo` — your central task list.

> This menu item only appears on folders that have default files configured by the blueprint.
