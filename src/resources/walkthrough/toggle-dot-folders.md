## Hide Configuration Folders

Memoria stores its config in `.memoria/` — but you probably don't need to see it day-to-day.

Use **Toggle dot-folders** to hide it:

1. Open the **Command Palette** (`Ctrl+Shift+P`)
2. Run **Memoria: Toggle dot-folders**

Run the command again to show the folders. Memoria only hides folders it manages — your own `files.exclude` settings are never touched.
