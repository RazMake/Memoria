# Media Assets

This folder is for GIF animations and screenshots used in the User Guide.

## Required files

| File | Shows |
|------|-------|
| `initialize-workspace.gif` | Command Palette → blueprint selection → scaffolded result |
| `initialize-multi-root.gif` | Same flow in a multi-root workspace |
| `toggle-dot-folders.gif` | Before/after dot-folder visibility in Explorer |
| `manage-features.gif` | Enabling/disabling features via the picker |
| `open-default-files.gif` | Right-click → files open side by side |

## Recording tips

- Use [ScreenToGif](https://www.screentogif.com/) on Windows
- Target resolution: ~800×500px
- Keep GIFs under 2MB each to avoid bloating the `.vsix`
- Use a clean VS Code theme (e.g., Dark Modern) for consistency
