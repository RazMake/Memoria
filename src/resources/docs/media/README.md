| `manage-features.gif` | Enabling/disabling features via the picker |
| `open-default-files.gif` | Right-click → files open side by side |
| `sync-tasks.gif` | Running Sync Tasks command with progress and collector update |

| `contacts-sidebar.gif` | Browsing, searching, and adding contacts in the sidebar |

| `todo-editor.gif` | Visual task board: cards, drag-drop, check/uncheck, add task |

| `snippets-contact.gif` | @-trigger contact snippets with format picker and hover |
| `snippets-copy-child.gif` | {copy-child} snippet duplicating a sub-heading section |
| `conflict-resolver.gif` | Reinitialization conflict resolution with diff editor |

## Recording tips

- Use [ScreenToGif](https://www.screentogif.com/) on Windows
- Target resolution: ~800×500px
- Keep GIFs under 2MB each to avoid bloating the `.vsix`
- Use a clean VS Code theme (e.g., Dark Modern) for consistency
