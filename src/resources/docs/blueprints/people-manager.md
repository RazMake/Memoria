# People Manager Notebook

*For managers: meeting notes, 1:1s, team contacts, and project status.*

## Folder Structure

```
📁 00-ToDo/
   📄 Main.todo          ← default file
   📄 test.md            ← default file
📁 01-People/
📁 02-ToRemember/
📁 03-MeetingNotes/
   📁 1-1/
   📁 Ad-hoc/
   📁 WOR/
📁 04-Inbox/
📁 05-Archive/
```
