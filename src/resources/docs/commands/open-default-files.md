# Open default file(s)

**Command:** `Memoria: Open default file(s)`
**Available:** Right-click context menu on folders with configured default files

Opens one or more pre-configured files for a folder, side by side in the editor. This provides quick access to the files you use most in each folder.

1. Right-click a folder in the Explorer that has a default file configured
2. Select **Memoria: Open default file(s)**
3. If any open files have unsaved changes, a checkbox list appears so you can choose which files to save — unchecked files are discarded
4. All current editors are closed and the default files open side by side

![Open default files](../media/open-default-files.gif)

> **Tip:** Default files are defined in the blueprint template and configured in `.memoria/default-files.json`. See [Default Files](../features/default-files.md) for details.
